# Changelog

## 0.1.0

- Initial release.
- Added `createConstructivePiExtension` factory for PI extension integration.
- Added GraphQL operation allowlist -> PI tool adapter.
- Added policy evaluation + inline approval handling.
- Added session-scoped auth token override command.
- Added env-driven extension entrypoint and operation file loader.
- Added operation schema helper (`createOperationParametersSchema`).
- Added SDK integration test (`createAgentSession` + `DefaultResourceLoader` + extension tools).
- Added usage examples for local `-e` and `pi install` package flows.
